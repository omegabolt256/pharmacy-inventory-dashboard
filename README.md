# pharmacy-inventory-dashboard
 No-code pharmacy inventory automation built in Airtable
A zero-code inventory management system built in Airtable to track 
medication expiration dates, reduce financial loss, and ensure patient safety.

## Problem it solves
Pharmacies lose money and risk patient safety when expired medications 
stay on shelves. Manual tracking is error-prone. This dashboard automates 
the entire monitoring and alert process.

## Features
- Auto-classifies every drug as Active, Expires Soon, or Expired
- 25 medications tracked with batch numbers and quantity
- Filtered "Action Required" view shows only drugs needing removal
- Daily email alert at 12pm sent to pharmacy staff automatically

## Tools used
- Airtable (database + formula engine + automations)
- 100% free, zero code written
